Torrey Pines Software, Inc. is a California corporation founded in 1987. Throughout its history, it has provided consulting and contracting services to both small and large corporations in the areas of:
  • PC and embedded systems software architecture, design, and coding
  • Systems-oriented consulting for both vertical and horizontal applications
  • Software project management and mentoring

Additionally, Torrey Pines has created and marketed various products, including database query and extraction tools and various system utilities.

Barry Demchak is currently the sole consultant at Torrey Pines. He has been active in the computer industry since 1971, having acquired technical expertise in a number of areas, including:

  • Java, C++, C, Visual Basic, Pascal, and Assembly Language
  • Intel PC-class processors and Dallas Semiconductor TINI
  • Windows XP/2000/NT, Windows 9x, OS/2, MS-DOS
  • System-level software, including operating systems, device drivers, compilers, graphical systems, networking, and communications
  • Audio editing workstations
  • Client-server applications
  • Micro-to-mainframe communications
